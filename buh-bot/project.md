# Бот управления бухгалтерским аутсорсингом

## О проекте

Telegram/Max-бот для автоматизации работы компании, оказывающей услуги бухгалтерского, кадрового, налогового и воинского учёта на аутсорсинге. Компании-клиенты расположены по всей России.

## Бизнес-контекст

Компания обслуживает клиентов в формате аутсорсинга:
- **Бухгалтерский учёт** (ОСН, УСН, ЕСХН, ПСН)
- **Кадровый учёт**
- **Налоговый учёт и отчётность**
- **Воинский учёт**

Коммуникация ведётся через мессенджеры **Max (90%)** и **Telegram (10%)**.  
У каждого клиента есть своя группа с бухгалтером и представителем клиента.  
У каждой компании-клиента есть **1С:Предприятие**.

## Проблемы, которые решает бот

| Проблема | Решение |
|---|---|
| Пропуск сроков сдачи отчётности | Автоматические напоминания за 14/7/3/1 день |
| Нет контроля над бухгалтерами | Задачи с дедлайнами, статусами, KPI |
| Доп. работы не считаются | Журнал доп. работ → таблица в конце месяца |
| Непонятно что входит в абонплату | Стандарт работы закреплён за каждой компанией |
| Нет оценки продуктивности | KPI: % выполнения, просрочки, ошибки, часы |
| Бухгалтеры не видят общую картину | Сводки в общую группу бухгалтеров |

## Модели данных

### companies — компании-клиенты
- `name`, `inn`, `tax_system`, `org_type`
- `has_employees`, `has_military`
- `max_group_id` — ID группы в Max для уведомлений
- `accountant_id` — закреплённый бухгалтер
- `work_standard` — текстовый стандарт базового пакета услуг

### accountants — сотрудники
- `name`, `max_user_id`

### report_deadlines — дедлайны отчётности
- `report_name`, `report_type`, `due_date`, `period`
- `status` (pending / done / overdue)
- Автогенерируются по налоговому календарю РФ

### tasks — задачи
- `title`, `description`, `due_date`, `priority`, `status`
- Привязаны к компании и бухгалтеру

### additional_works — дополнительные работы
- `work_type`, `description`, `hours`, `amount`, `work_date`
- Попадают в Excel-отчёт в конце месяца

### accountant_errors — журнал ошибок
- `accountant_id`, `company_id`, `description`, `error_date`
- Учитываются в KPI

## Роли пользователей

| Роль | Как определяется | Доступ |
|---|---|---|
| **Руководитель** | max_user_id в ADMIN_IDS (.env) | Все функции |
| **Бухгалтер** | Запись в таблице accountants | Свои задачи и дедлайны |
| **Клиент** | Участник группы | Видит статус через /status |

## Функционал по ролям

### Руководитель
- Добавление/редактирование/деактивация компаний
- Добавление бухгалтеров
- Постановка задач
- Добавление стандартных и произвольных дедлайнов
- Просмотр всех дедлайнов, просроченных, KPI
- Фиксация ошибок бухгалтеров
- Excel-отчёт доп. работ + KPI за месяц

### Бухгалтер
- Просмотр своих задач и дедлайнов
- Отметка выполнения (кнопкой)
- Фиксация доп. работ

### Автоматика (планировщик, 9:00 МСК)
- Напоминания в группы клиентов (за 14/7/3/1 день)
- Личные напоминания бухгалтерам
- Сводка просроченных — администраторам и в группу бухгалтеров
- Утренняя сводка дедлайнов на 7 дней — в группу бухгалтеров

## Стандарт работы vs. доп. работы

**Базовый пакет** (входит в абонплату, прописывается в карточке компании):
- Ведение бухучёта
- Расчёт зарплаты
- Кадровое делопроизводство (стандарт)
- Плановая отчётность по налоговому календарю
- Воинский учёт (если включён)

**Доп. работы** (фиксируются отдельно, оплачиваются сверх абонплаты):
- Восстановление учёта
- Ответы на требования ФНС/СФР
- Подготовка документов для банка/суда
- Представительство в ФНС
- Камеральные/выездные проверки
- Развёрнутые консультации (>30 мин)
- Кадровые документы сверх стандарта
- Регистрация / ликвидация / изменения
- Исправление ошибок прошлых периодов

## Excel-отчёт (ежемесячный)

**Лист 1 — Доп. работы:**
- Разбивка по клиентам
- Дата, тип, описание, часы, сумма, исполнитель
- Итог по каждому клиенту и общий итог за месяц

**Лист 2 — KPI бухгалтеров:**
- Количество компаний
- Дедлайнов всего / выполнено / просрочено
- % выполнения (исполнительность)
- Количество ошибок / коэффициент ошибок
- Доп. часы / доп. сумма
- Просроченные и ошибки подсвечены красным

## Налоговый календарь РФ (автогенерация)

При добавлении компании система автоматически генерирует все стандартные дедлайны на текущий год:

**ОСН:** НДС (ежеквартально, 25-е), налог на прибыль, имущество, 6-НДФЛ, РСВ, ЕФС-1, БО  
**УСН:** декларация, авансы, 6-НДФЛ/РСВ/ЕФС-1 (если есть сотрудники)  
**ЕСХН:** декларация, аванс, сотрудники  
**Сотрудники:** НДФЛ (два срока в месяц), страховые взносы ежемесячно  
**Воинский учёт:** форма №6, список военнообязанных, сведения о гражданах

Все даты автоматически переносятся на ближайший рабочий день если выпадают на выходные.
