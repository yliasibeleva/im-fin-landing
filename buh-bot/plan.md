## Этап 2 — Веб-интерфейс (В ПЛАНАХ)

### Цель
Удобный веб-интерфейс для руководителя: таблицы, фильтры, экспорт — без Telegram/Max.

### Технический стек (предварительно)
- **Backend:** FastAPI (уже используется Python)
- **Frontend:** простой HTML/Bootstrap или React (лёгкий)
- **БД:** та же SQLite / миграция на PostgreSQL
- **Авторизация:** простой токен в .env или Basic Auth

### Функционал этапа 2

#### Дашборд руководителя
- [ ] Сводная таблица всех компаний
- [ ] Дедлайны текущей недели / месяца с фильтрами
- [ ] Статус по бухгалтерам (загруженность)
- [ ] Просроченные — красным, выделение

#### Управление компаниями (веб)
- [ ] Список с поиском и фильтром по бухгалтеру / СНО
- [ ] Полная карточка со стандартом работы
- [ ] Редактирование через форму
- [ ] История изменений

#### Управление дедлайнами (веб)

- [ ] Массовое добавление дедлайнов из шаблона

#### Задачи (веб)

### Рекомендуемые провайдеры (РФ)

- **Selectel** — от 350 руб/мес
- **Reg.ru** — от 249 руб/мес

### Шаги деплоя
```bash
# На сервере (Ubuntu 22.04)
git clone <repo> /opt/buh-bot
cd /opt/buh-bot
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
cp .env.example .env
nano .env  # заполнить токен и ID

# Systemd-сервис
sudo nano /etc/systemd/system/buh-bot.service
sudo systemctl enable buh-bot
sudo systemctl start buh-bot
```

### Содержимое systemd-сервиса
```ini
[Unit]
Description=Buh Bot
After=network.target

[Service]
Type=simple
WorkingDirectory=/opt/buh-bot
ExecStart=/opt/buh-bot/venv/bin/python main.py
Restart=always
RestartSec=10

[Install]
WantedBy=multi-user.target
```
